# Working Directory

## Objective

Display the current working directory in Linux.

---

## Command Used

```bash
pwd
```

---

## Sample Output

```text
/home/kali
```

---

## Explanation

The `pwd` (Print Working Directory) command displays the absolute path of the directory in which the current terminal session is operating.

A working directory is the folder where files and folders are created, modified, copied, or deleted unless another path is explicitly specified.

---

## Why this Command is Useful

- Displays the current location in the filesystem.
- Prevents accidental operations in the wrong directory.
- Useful while writing scripts.
- Helps users navigate large directory structures.

---

## Cybersecurity Relevance

During security assessments, analysts constantly navigate through:

- Log directories
- Configuration files
- Evidence folders
- Exploit repositories

Knowing the current directory helps avoid mistakes while handling sensitive files.

## Conclusion

The `pwd` command helps users understand their current location in the Linux filesystem, making navigation and file management easier.
