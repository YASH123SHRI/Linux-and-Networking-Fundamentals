# Files and Directories

## Objective

Create a new directory named **CyberLab** and understand the difference between files and directories.

---

## Command Used

```bash
mkdir CyberLab
```

---

## Verification

```bash
ls
```

---

## Sample Output

```text
CyberLab
```

---

## Explanation

The `mkdir` command is used to create a new directory.

In this exercise, a directory named **CyberLab** was created to organize project files.

---

## File vs Directory

| File | Directory |
|------|-----------|
| Stores actual data | Stores files and subdirectories |
| Contains information such as text, images or programs | Organizes related files into folders |
| Cannot contain another file inside itself | Can contain multiple files and directories |

---

## Why Directories are Important

Directories help:

- Organize files efficiently.
- Separate projects.
- Improve filesystem management.
- Reduce clutter.

---

## Cybersecurity Relevance

Security analysts often maintain directories for:

- Evidence collection
- Log files
- Scan results
- Malware samples
- Scripts
- Reports

A structured directory layout makes investigations easier.

## Conclusion

Directories provide an organized structure for storing files and are essential for maintaining large cybersecurity projects.
